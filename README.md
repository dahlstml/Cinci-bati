Cinci-BATi 🦇
Cincinnati's Bats: Eight Species, One Fight
An interactive digital StoryMap built to educate the general Cincinnati public about Indiana bat and local bat species conservation. This project was created as part of a science communication course final project.
What's inside:

Species profiles for all 8 bat species found in the Cincinnati area
Interactive quiz on bat facts
Threat overview covering White Nose Syndrome, habitat loss, and accidental harm
Original illustrated bat habitat and threat map of the Cincinnati tri-state area
Five actionable steps Cincinnati residents can take to help
Full citations from USFWS, peer-reviewed research, and conservation organizations
